MC开服教程

发表于2023-04-11|更新于2023-04-11|MC
|阅读量:

前言

现在各位小伙伴们应该已经开学了，玩 MC 的时间也不多。但是又特别想要和小伙伴们联机玩怎么办呢？一种办法是使用内网穿透，但是需要腐竹一直在线。如果腐竹不在线的话，那也需要腐竹的电脑一直挂着 MC，这显然有点麻烦。

所以，拥有一台服务器来开 MC 对提升 MC 联机的体验还是很有必要的～

但是，大家都知道，一般的云服务器都很昂贵。小伙伴们可以自己去腾讯云、阿里云这种顶级大厂首页看看，虽然新人购买服务器有优惠，但是优惠的服务器不能满足开 MC 的需要。如果要满足需求的话，再怎么新人都得上千元。这显然很矛盾啊 = =

可能有小伙伴想要去淘宝买面板服，可是那里的面板服可能是超开的，也就是 “能不配价”，而且售后不一定得到保障。

那怎么办呢？

雨云游戏云

其实夜梦这里倒是有一个推荐，那就是：

雨云游戏云

当然了，夜梦绝对不会随便推荐的，推荐雨云肯定是有理由的。夜梦将推荐理由列在下面供大家参考：

  • 对比普通云服务器，游戏云具有更高的性能优势。雨云的游戏云提供三种规格的 CPU，大家可以根据自己的 MC 版本和模组情况选择最适合自己的 CPU 以节省费用。具体 CPU 型号分别是：
  • 13900K 5.8Ghz
  • 5900X 4.8Ghz
  • E5 2666v3 3.4Ghz* 雨云游戏云提供 VPS 和面板服两种选择。VPS 具有高性能并且拓展性极强；面板服则简单易上手，不超开保证质量。
  • 面板服支持日付，最低每日仅需要一包辣条的价格。并且提供备份，不想玩的时候只要将地图数据备份到本地，然后删除服务器即可停止计费，周末有空的时候可以上传自己的地图再次开服！
  • 游戏云提供 “电量计费”，足以在保证服务器流畅度的同时再一次降低开服的费用！

购买游戏云

雨云直达链接：https://www.rainyun.cc

注册时优惠码填写 MTk2Njk= 可以享受五元无条件减免的优惠哦！

购买方式十分简单，在主页选择游戏云：


点击购买游戏云：


这里提供了翼龙面板和 MCSM 面板两种面板，小伙伴可以根据自己的喜好进行选择哦！夜梦这里选择的是翼龙面板～

选择自己心仪的配置之后，点击立即购买即可！

PS：VPS 没有按日计费哦！

配置游戏云

查看配置

夜梦这里拿翼龙面板进行操作：

我们先在进入游戏云的后台：

可以在雨云的后台查看服务器的具体信息：

  • 夜梦用绿色框框框出来的就是我们的 “电量”。雨云游戏云价格便宜的原因就是他们使用电量来结算 CPU 使用费用。具体可以参考雨云的官方计费文档哦：forum.rainyun.com/t/topic/648
  • 夜梦用蓝框框出来的就是用余额结算电费，如果你开启的话，当你的 CPU 电量没有以后，雨云会使用你的余额等价抵扣超量 CPU 产生的费用。如果你担心熊孩子捣乱的话，可以关闭哦～后期如果缺电量的话，可以使用积分兑换 CPU 电量，也十分方便！
  • 下面夜梦用红框框框出的是是否按日计费。雨云的游戏云支持按日计费，入门版每日仅需要 0.9 元，如果你一时没有积分免费续费了，也可以每日消费 0.9 元进行续费，而且续费还会有积分返利，这样你又可以免费续费了～
  • 如果担心熊孩子捣乱，那就关闭 “用余额结算电费”。
  • 如果不想接着开服了，那就关闭 “按日计费”。

切换游戏版本

如果想要切换游戏版本的话，操作很简单：

点击控制台上方的：重装 / 切换游戏

然后就可以自己选择版本啦！记得勾选确认操作并放弃服务器内数据哦！

进入面板

找到我们的服务器，点击进入：

然后就可以看到具体的开服界面了：

配置 MC 服务器

我们已经在 _4.2 切换游戏版本_中选定了自己想要的 MC 服务器版本，一般情况下，雨云已经在后台为你添加了这个版本的 MC 服务器核心并且配置好了启动指令。我们要做的只要开机就好了！

如果想要上传自己的服务器核心，只要到文件中删除所有的文件然后上传自己的文件就好了！启动指令可以在启动中修改：

开启服务器并进入游戏

开启服务器

点击开机按钮以后，等待一段时间，看到如下提示时就说明已经启动成功了：

点击地址即可复制 MC 服务器地址：

进入游戏服务器

服务器默认开启正版验证，如果想要盗版也能进入，需要我们修改 sever-properties 中的 online-mode 属性。将其改为 false

结尾

到此，MC 服务器就开好啦！小伙伴们就可以进入服务器快乐地玩耍了！
